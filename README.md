# Kasturbai College of Education (KCEBED) Website

A modern, responsive, and dynamic website designed for **Kasturbai College of Education, Solapur**. This project serves as the digital front for the institution, providing information about admissions, courses, facilities, alumni, and student support.

**Live Demo:** [https://kcebed.onrender.com](https://kcebed.onrender.com)

## Features

*   **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
*   **Modern UI:** Clean interface using CSS Grid and Flexbox with a consistent color theme.

## Tech Stack

*   **HTML5:** Semantic structure.
*   **CSS3:** Custom styling with CSS Variables, Flexbox, and Grid layouts.
*   **JavaScript (Vanilla):** Dynamic DOM manipulation and component fetching.
